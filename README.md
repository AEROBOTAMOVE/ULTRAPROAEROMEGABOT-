## 🔒 Честно (не се крие)
- Backtest числа = минало; не гарантират бъдеще. **Хартия/малък размер първо.**
- LONG е доказан във всички епохи (вкл. мечи 2011-2015); SHORT е слаб исторически
  (печели само в мечи режим — ботът показва режима).
- Сигналите между таймфреймовете/металите се припокриват (едно макро ядро).
- **Не е финансов съвет.** Човекът на чарта решава.
